O que é uma String?

Imagine que você quer guardar a palavra "Hello" no computador. Em C, você não tem um tipo chamado "texto" — em vez disso, você usa uma sequência de letras (caracteres) guardadas uma do lado da outra na memória. Isso é uma string.

Todo string em C termina com um caractere especial invisível chamado terminador nulo ('\0'). Ele serve para o programa saber onde a palavra acaba.

A biblioteca

Para usar as funções de manipulação de strings, você precisa incluir essa biblioteca no topo do código: 

        
          #include

Como declarar e inicializar uma String ?

Existem duas formas principais:

Forma 1 — Tamanho fixo (você define o espaço):

          
            char mensagem[20]; // reserva espaço para até 20 caracteres

Forma 2 — Inicializando direto com um valor: 

          
            char mensagem[] = "Hello, world!";

Aqui o compilador conta sozinho quantas letras tem e reserva o espaço certo (incluindo o \0 no final). Muito mais prático!

Como acessar uma letra específica da String ?

Como a string é um array (uma lista), cada letra tem um número de posição (índice), começando do zero. 

         
            char palavra[] = "Código";

            char segundo_caractere = palavra[1]; // pega a letra na posição 1
            printf("Segundo caractere: %c\n", segundo_caractere);
            // Saída: ó

⚠️ Atenção: a contagem começa no 0, então palavra[0] = 'C', palavra[1] = 'ó', e assim por diante.

Percorrendo uma String com laço de repetição

E se você quiser mostrar cada letra da palavra, uma por linha? Usa um for junto com strlen() (que conta o tamanho da string): 

        
          #include 
          #include  // necessário para usar strlen

          int main() {
          char palavra[] = "Código";
          int cont;

          printf("A palavra contém: %i letras.\n", strlen(palavra));

          for (cont = 0; cont < strlen(palavra); cont++) {
          printf("Letra %i: %c\n", cont, palavra[cont]);
          }
          }
          ```

          **Saída:**
          ```
          A palavra contém: 6 letras.
          Letra 0: C
          Letra 1: ó
          Letra 2: d
          Letra 3: i
          Letra 4: g
          Letra 5: o

Funções de Manipulação de String

1️⃣ strlen() — Mede o tamanho da string

Retorna quantos caracteres a string tem (sem contar o \0). 

        
          char str[] = "Hello";
          printf("Length: %lu\n", strlen(str));
          // Saída: Length: 5

2️⃣ strcpy() — Copia uma string para outra

Funciona como um Ctrl+C / Ctrl+V de strings. 

        
          char source[] = "World";
          char destination[20];
          
          strcpy(destination, source);
          printf("%s\n", destination);
          // Saída: World

💡 Você não pode fazer destination = source diretamente em C. Por isso existe o strcpy.

3️⃣ strcat() — Junta (concatena) duas strings

Pega uma string e cola outra no final dela. 

        
          char str1[20] = "Hello, ";
          char str2[] = "World!";
          
          strcat(str1, str2);
          printf("%s\n", str1);
          // Saída: Hello, World!

⚠️ O str1 precisa ter espaço suficiente para receber o conteúdo de str2!

Exemplo prático — juntando nome e sobrenome: 

        
          char primeiro_nome[50];
          char sobrenome[50];
          char nome_completo[100];
          
          scanf("%s", primeiro_nome);
          scanf("%s", sobrenome);
          
          strcpy(nome_completo, primeiro_nome); // copia o primeiro nome
          strcat(nome_completo, " ");           // adiciona um espaço
          strcat(nome_completo, sobrenome);     // adiciona o sobrenome
          
          printf("Nome completo: %s\n", nome_completo);

4️⃣ strcmp() — Compara duas strings

Verifica se duas strings são iguais ou qual vem "antes" alfabeticamente.

•Retorna 0 se as strings forem iguais

•Retorna valor negativo se a primeira vem antes alfabeticamente

•Retorna valor positivo se a primeira vem depois 

          
            if (strcmp("apple", "orange") < 0) {
            printf("apple vem antes de orange\n");
            }

5️⃣ strncpy() — Copia apenas N caracteres

Igual ao strcpy, mas você define quantas letras quer copiar.

          
            char src[] = "World";
            char dest[20];
            
            strncpy(dest, src, 3);  // copia só 3 letras
            dest[3] = '\0';          // coloca o terminador nulo manualmente!
            
            printf("%s\n", dest);
            // Saída: Wor

⚠️ O strncpy não coloca o \0 automaticamente, então você precisa fazer isso manualmente.

6️⃣ strncat() — Concatena apenas N caracteres

Igual ao strcat, mas você define quantas letras do segundo string vão ser coladas. 

          
            char str1[20] = "Hello, ";
            char str2[] = "World!";
            
            strncat(str1, str2, 3); // cola só as 3 primeiras letras de str2
            printf("%s\n", str1);
            // Saída: Hello, Wor

7️⃣ strncmp() — Compara apenas N caracteres

Igual ao strcmp, mas compara apenas os primeiros N caracteres. 

          
            if (strncmp("casa", "caso", 3) == 0) {
                printf("Os primeiros 3 caracteres são iguais!\n");
            }
            // Saída: Os primeiros 3 caracteres são iguais!

Os 3 primeiros caracteres de "casa" e "caso" são "cas" — iguais! Por isso retorna 0.

8️⃣ strchr() — Encontra um caractere na string

Procura a primeira vez que um caractere aparece na string e retorna a posição dele. 

        
          char str[] = "Hello, World!";
          char *ptr = strchr(str, 'W'); // procura o 'W'
          
          if (ptr) {
              printf("Caractere encontrado: %c\n", *ptr);
              // Saída: Caractere encontrado: W
          }

Esse *ptr é um ponteiro

• Uma variável normal guarda um valor (ex: o número 10)

• Um ponteiro guarda um endereço de memória (ex: "o 'W' está na posição 7 da memória")

• O *ptr serve para ir até esse endereço e pegar o valor que está lá

9️⃣ strstr() — Encontra uma substring dentro de outra string

Procura uma palavra inteira (ou trecho) dentro de outra string. 

        
          char str[] = "Hello, World!";
          char *ptr = strstr(str, "World"); // procura "World" dentro de str
          
          if (ptr) {
              printf("Substring encontrada: %s\n", ptr);
              // Saída: Substring encontrada: World!
          }

Note que ele retorna tudo a partir de onde encontrou — por isso a saída é World! e não só World.

🔟 strtok() — Divide uma string em partes (tokens)

Quebra uma string em pedaços com base em um separador (como vírgula, espaço, etc.).

        
          char str[] = "Hello, World!";
          char *token = strtok(str, ", "); // separa pelo espaço e vírgula
          
          while (token != NULL) {
              printf("%s\n", token);
              token = strtok(NULL, ", "); // continua de onde parou
          }
          
          // Saída:
          // Hello
          // World!

💡 Na primeira chamada você passa a string. Nas chamadas seguintes, passa NULL para continuar de onde parou.
